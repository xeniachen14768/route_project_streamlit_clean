\# Traffic Light Route Planning System



This project is an interactive route planning system based on OpenStreetMap, OSMnx, NetworkX, Folium, and Streamlit.



The system allows users to select a start point and a goal point directly on the map. It then calculates and compares three routes:



1\. Shortest Path

2\. Alternative Path

3\. Fastest Path considering simulated traffic light delay



\## Features



\- Load OpenStreetMap `.osm` map data

\- Detect traffic light nodes from OSM data

\- Simulate traffic light cycle, green time, and offset

\- Allow users to select start and goal points on the map

\- Convert selected coordinates to nearest road nodes

\- Calculate shortest route by distance

\- Generate an alternative route by penalizing the shortest path

\- Calculate fastest route considering:

&#x20; - vehicle speed

&#x20; - departure time

&#x20; - traffic light waiting time

\- Display route comparison table

\- Visualize routes on an interactive map



\## Route Types



| Route | Description |

|---|---|

| Shortest Path | The route with the shortest travel distance |

| Alternative Path | A substitute route generated by penalizing the shortest route |

| Fastest Path | The route with the lowest total travel time considering traffic light delay |



\## Map Legend



\### Markers



\- Start Point: green marker

\- Goal Point: blue marker

\- Traffic Lights: red circle markers



\### Routes



\- Shortest Path: red solid line

\- Alternative Path: blue dashed line

\- Fastest Path: green solid line
